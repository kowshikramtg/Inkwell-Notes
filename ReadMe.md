# Inkwell Notes

A modern, efficient, and user-friendly note-taking application.

</div>

---

## Overview

Inkwell Notes is a lightweight yet powerful note-taking application built to enhance productivity and simplify information management. It provides a clean interface, intuitive features, and reliable performance for everyday use.

---

## Features

- Create, edit, and delete notes easily  
- Clean and minimal user interface  
- Fast and responsive performance  
- Secure and structured data handling  
- Search and organize notes efficiently

---

## Tech Stack

- Frontend Framework: Angular
- language: TypeScript
- UI Structure: Component-based Architecture
- State Management: Angular Services
- Data Handling: Local Models
- Version Control: Git & GitHub
- Architecture Pattern: Modular & Component-Driven Design

---

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kowshikramtg/Inkwell-Notes.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Inkwell-Notes
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the application:
   ```bash
   npm start
   ```

---

## Usage

- Open the application in your browser  
- Create a new note  
- Edit or delete notes  
- Use search to find notes quickly
